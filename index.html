Features

Everything you need,
nothing you don't

Built for developers who want powerful AI assistance without the overhead of cloud-only tools.

Local-First Privacy

Run models directly on Apple Silicon with MLX. Your code never leaves your machine. Zero data sent to cloud unless you choose a cloud provider.

Multi-Provider Support

Connect to Anthropic, OpenAI, Groq, Google, DeepSeek, Mistral, Together, OpenRouter, and more. Switch providers with a single click.

Agent System

Autonomous AI agents that execute multi-step tasks, use tools, and iterate on code. Define custom skills and let agents handle complex workflows.

Auto-Model Routing

Describe your task and Clew picks the best model. Fast models for simple queries, powerful models for complex reasoning. Automatic optimization.

Code Intelligence

Built-in LSP client for real-time code completion, diagnostics, and navigation. Supports any language with an LSP server.

Native Desktop App

Built with Qt/PySide6 for a truly native experience on macOS and Windows. No Electron. No web wrapper. Fast, light, and responsive.

Integrations

Works with every major AI provider

One interface for all your AI models. Switch providers instantly, compare outputs, and route tasks automatically.

Get Started

Up and running in under a minute

No complex setup. Download, add your API key, and start coding with AI assistance.

Download & Install

Get Clew for macOS (DMG) or Windows (EXE). Native install, no dependencies required. Just double-click and go.

Add Your API Key

Open Settings and enter your API key for any supported provider. Or use local models with zero configuration on Apple Silicon.

Start Building

Open a project, chat with AI, use agents for complex tasks, and get real-time code completions powered by LSP.

Built for Developers

Powerful, yet simple to use

Agent system, token tracking, and model routing work together to give you the best AI-assisted coding experience.

agent_config.json
// Define a custom agent skill { "name": "code-reviewer", "model": "claude-sonnet-4-20250514", "tools": [ "read_file", "search", "edit_file", "run_tests" ], "auto_route": true, "max_tokens": 8192, "system": "You are a senior code reviewer..." }

Custom Agent Skills

Define specialized agents with their own tools, system prompts, and model preferences. Build a team of AI assistants.

Real-Time Token Tracking

Monitor token usage per provider, per session. Automatic context management prevents unexpected cutoffs.

Intelligent Model Routing

Describe what you need. Clew analyzes the task complexity and routes it to the most cost-effective model.
